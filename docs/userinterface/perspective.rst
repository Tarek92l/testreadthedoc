============
Persepective
============

A perspective defines the initial set and layout of views in the
Workbench window. Within the window, each perspective shares the same
set of editors. Each perspective provides a set of functionality aimed
at accomplishing a specific type of task or works with specific types of
resources.

To open the “Platform” perspective to get the dedicated views for the
System Workbench projects. To open this perspective, go to *Window >
Open Perspective > Other...* then select *Ac6 Platform*.
