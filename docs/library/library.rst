============
Ac6 Library
============

About the library
------------------
Ac6 Library is a set of plugins to support the Platforms and package definitions library. A library consists on a set of description files which describes preconfigured Linux packages definitions which can be added to a Platform project. It also provide a set of default and ready-to-use Platform project which can be imported into your workspace
